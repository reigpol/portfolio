# Portfolio — Pol Reig Plazas

Personal portfolio of an Odoo Tech Lead & Software Engineer. Built with vanilla HTML, CSS, and JavaScript.

## Preview

Open `index.html` in any modern browser. No build step or dependencies required.

## Structure

```
├── index.html   # Semantic markup
├── styles.css   # Custom properties, fluid typography, CSS animations
└── script.js    # IntersectionObserver, custom cursor, mobile nav
```

## Highlights

- Custom cursor with hover states
- Scroll-triggered reveals via IntersectionObserver
- Magnetic button effects
- Active section tracking in sidebar
- Mobile-first responsive layout
- Respects `prefers-reduced-motion`
