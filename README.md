# Portfolio — Pol Reig Plazas (Gemini 3.5 Flash Edition)

Personal portfolio of an Odoo Tech Lead & Software Engineer. Built with vanilla HTML, CSS, and JavaScript, featuring a custom technical-editorial aesthetic and interactive core components.

## Preview

Open `index.html` in any modern browser. No build steps or dependencies are required.

## Structure

```
├── index.html          # Semantic markup with complete English resume data
├── style.css           # Custom properties, volcanic-amber dark mode, fluid typography, blueprint detailing
├── app.js              # Theme manager, modular ERP sandbox simulator, scroll triggers, clipboard interactions
├── CVPolReigPlazas.pdf # Downloadable curriculum vitae (PDF)
└── LICENSE             # MIT License
```

## Highlights

- **Bespoke Warm Editorial Theme**: Beautiful light cream ivory coupled with an ultra-premium volcanic basalt, molten amber & rose-orchid dark mode.
- **Architectural Detailing**: Customized neon-brutalist blueprint corner joints (`+` markers) on key content layouts in dark mode.
- **Adaptive Header & Navigation**: Fully responsive navigation containing a fluid theme switch with persistent preference mapping (`localStorage`) and glassmorphic backdrop filters.
- **Interactive ERP Sandbox**: Live module visualization simulator mapping code blocks (Python & FastAPI schemas) to active system diagrams on tap.
- **Scroll-Triggered Reveals**: Smooth staggered fade-in visual reveals leveraging high-performance `IntersectionObserver` states.
- **Tactile Mobile Tabs**: Ergonomic mobile tab navigations for custom ERP widgets, maximizing swipe efficiency.
